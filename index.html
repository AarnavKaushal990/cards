HTML Basics

HTML stands for HyperText Markup Language. It is used to structure content on the web. Basic elements include <div>, <span>, <a>, and <img>.

CSS Basics

CSS stands for Cascading Style Sheets. It is used to style and layout web pages. Common properties include color, background-color, font-size, and margin.

JavaScript Basics

JavaScript is a programming language used to create interactive effects on web pages. Basic syntax includes variables (let, const), functions, and control structures (if, for).

HTML Tags

HTML uses tags to define elements. Common tags include <h1> for headings, <p> for paragraphs, and <ul> for unordered lists.

CSS Selectors

CSS selectors are used to apply styles to elements. Examples include .class for class selectors, #id for ID selectors, and element for element selectors.

JavaScript Events

JavaScript can respond to user interactions through events such as click, mouseover, and keydown. Event listeners can be added using addEventListener().

HTML Forms

HTML forms are used to collect user input. Key elements include <form>, <input>, <textarea>, and <button>.

CSS Flexbox

CSS Flexbox is a layout model that allows you to design complex layouts with ease. Key properties include display: flex, justify-content, and align-items.

JavaScript DOM Manipulation

The Document Object Model (DOM) can be manipulated with JavaScript to change the structure and content of web pages. Methods include getElementById(), querySelector(), and innerHTML.

HTML Attributes

HTML attributes provide additional information about an element. Examples include href for links, src for images, and alt for alternative text.

CSS Box Model

The CSS Box Model describes the rectangular boxes generated for elements. It includes margins, borders, padding, and content.

JavaScript Variables

JavaScript variables are used to store data values. Use let or const to declare variables, and var for older code.

HTML Media Elements

HTML media elements include <audio> and <video> for embedding multimedia content.
